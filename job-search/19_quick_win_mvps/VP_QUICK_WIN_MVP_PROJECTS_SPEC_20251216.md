## Project #4: Interview Prep Chatbot
**Build Time:** 2 weeks
**Valuation:** $300K-800K
**Year 1 Revenue:** $50K-150K

### What It Does:
AI interview coach that conducts realistic mock interviews with personalized feedback and improvement tracking.

### Key Features:
- **Role-Specific Practice:** User selects job (frontend dev, PM, designer, etc.)
- **Realistic Conversation:** AI asks follow-up questions, challenges answers
- **Speech-to-Text:** Practice speaking answers aloud (not just typing)
- **Instant Feedback:** "Your answer was too generic. Add specific metrics from your GenCraft project."
- **Progress Tracking:** See improvement over 10+ practice sessions
- **Company Research:** AI studies target company (LinkedIn, Glassdoor) to ask relevant questions

### Tech Stack:
- **Frontend:** React + Web Speech API (voice input/output)
- **AI:** OpenAI API (GPT-4 with interview coaching system prompts)
- **Voice:** ElevenLabs or Google Cloud TTS (realistic interviewer voice)
- **Backend:** Supabase (session history, feedback, progress tracking)
- **Deployment:** Vercel

### User Workflow:
1. User creates profile (resume upload, target role, target companies)
2. Selects interview type (behavioral, technical, system design, etc.)
3. Clicks "Start Interview" → AI interviewer asks first question via voice
4. User responds aloud (speech-to-text captures answer)
5. AI follows up with clarifying questions (just like real interview)
6. After 30-45 minutes, AI provides detailed feedback report
7. User reviews feedback, tracks improvement over sessions

### Monetization:
- **Free Tier:** 1 practice session, basic feedback
- **Pro:** $39/month - Unlimited sessions, detailed feedback, progress tracking
- **Premium:** $99/month - Company-specific prep, resume review, offer negotiation coach

### Market Validation:

- **Market Size:** 10M+ job seekers in tech annually
- **Competitors:** Pramp (peer practice, not AI), InterviewCake ($249 one-time, no AI coach)
- **Unique Angle:** AI coach available 24/7, unlimited practice, fraction of cost

### Revenue Potential:

## Project #5: Chrome Extension - PDF Watermarker
**Build Time:** 1 week
**Valuation:** $200K-500K
**Year 1 Revenue:** $20K-50K

### What It Does:
One-click PDF watermarking for freelancers and consultants to protect proposals, contracts, and portfolios from unauthorized sharing.

### Key Features:
- **Browser Extension:** Right-click any PDF → "Add Watermark" → Done
- **3 Watermark Types:**
  - Simple text overlay ("CONFIDENTIAL - Jane Doe")
  - Email-triggered forensic (reveals recipient email when forwarded)
  - Invisible fingerprint (traces leaks to source)
- **Batch Processing:** Watermark entire folders at once
- **Custom Templates:** Save watermark presets for different clients
- **Analytics Dashboard:** See when/where PDFs are opened (tracking pixels)

### Tech Stack:
- **Extension:** Chrome Extension Manifest V3 (JavaScript)
- **PDF Processing:** PDF.js + custom watermarking algorithm (borrowed from TokenCalc)
- **Backend:** Supabase (watermark templates, analytics)
- **Deployment:** Chrome Web Store

### User Workflow:
1. Install Chrome extension
2. Right-click any PDF in browser or file system
3. Select watermark type (text, forensic, or invisible)
4. Click "Apply Watermark" → New PDF generated instantly
5. Analytics dashboard shows when PDF is opened (if tracking enabled)

### Monetization:
- **Free Tier:** 5 watermarks/month, text-only
- **Pro:** $9/month - Unlimited, all watermark types, analytics
- **Enterprise:** $49/month - Team licensing, API access, white label

### Market Validation:
- **Problem:** Freelancers lose $10K-50K/year to proposal plagiarism (competitors steal ideas)
- **Market Size:** 60M+ freelancers globally (Upwork, Fiverr, independent)
- **Competitors:** Adobe Acrobat ($20/mo, overkill for simple watermarking)
- **Opportunity:** Lightweight, affordable, freelancer-focused tool

### Revenue Potential:
