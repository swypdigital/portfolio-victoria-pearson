**Key Files:**

- `/src/agents/` - 12 specialized agents for different generation tasks
- `/src/patterns/` - 35 reusable ATP (Agentic Testing Patterns)

**Setup Instructions:**
```bash
# Clone repo
git clone https://github.com/[VICTORIA]/gencraft-v3.0
cd gencraft-v3.0

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add: ANTHROPIC_API_KEY, SUPABASE_URL, SUPABASE_ANON_KEY

# Run development server
npm run dev

# Run tests (100% pass rate required)
npm test

# Deploy to production
vercel --prod
```

**Common Issues:**
1. **API Rate Limits:** GenCraft can hit Anthropic's rate limits if generating >10 projects/hour
   - Solution: Implement queue system with 5-second delays
2. **Database Locks:** Supabase can lock during heavy writes
   - Solution: Use read replicas for queries
3. **Build Timeouts:** Large projects (10,000+ files) can timeout on Vercel
   - Solution: Switch to Railway for backend (no timeouts)
