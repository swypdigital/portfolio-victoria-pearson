# Tech Innovator Bright - Theme Summary

**Portfolio Scheme 4** | Infrastructure/DevOps Specialist Theme

## Theme Overview
Vibrant, energetic design optimized for Infrastructure and DevOps roles. Bright color palette with modern card-based layouts, pronounced shadows, and optimistic interactive states.

## Color Palette

### Primary Colors
- **Vibrant Blue**: #0066CC (primary), #3385DB (light), #0052A3 (dark)
- **Energetic Orange**: #FF6600 (secondary), #FF8533 (light), #CC5200 (dark)

### Accent Colors
- **Teal**: #00A896 (base), #00C9B7 (light)
- **Purple**: #8B5CF6 (base), #A78BFA (light)

### Background Colors
- **Light Grey**: #FAFAFA (primary background)
- **Off-White**: #F5F5F5 (secondary background)
- **Pure White**: #FFFFFF (cards)

### Text Colors
- **Dark Grey**: #333333 (primary text)
- **Medium Grey**: #666666 (secondary text)
- **Light Grey**: #999999 (tertiary text)

## Typography
- **Primary Font**: Poppins (headings, navigation)
- **Secondary Font**: Work Sans (body text)
- **Weights**: 400 (regular), 500 (medium), 600 (semi-bold), 700 (bold), 800 (extra-bold)

## Design Features

### Card-Based Layouts
- **Shadows**: 4 levels (sm, md, lg, xl) for depth
- **Border Radius**: 8px (sm), 12px (md), 16px (lg)
- **Colorful Borders**: 3px left/top borders with gradient colors

### Interactive States
- **Hover Lift**: translateY(-4px)
- **Hover Scale**: scale(1.02)
- **Transition Speed**: 0.2s (fast), 0.3s (medium), 0.5s (slow)

### Modern Elements
- **Gradient Text**: Multi-color gradients for headings
- **Icon Integration**: Ready for tech stack icons
- **Colorful Tags**: Skill and tech tags with hover effects
- **Timeline Design**: Gradient vertical line with colored cards
